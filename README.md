# plupdate-db

Community database of AAX plugin metadata for the
[Plupdate](https://github.com/bootlegcheeseburger/plupdate) macOS app.

## Contribute

Spotted a wrong version or missing plugin? Edit the relevant
[`db/vendors/<vendor>.json`](db/vendors/) on GitHub - pencil icon, propose
changes, CI validates. Full guide in
[`db/CONTRIBUTING.md`](db/CONTRIBUTING.md).

## Data URL

```
https://bootlegcheeseburger.github.io/plupdate-db/vendors/index.json
```
